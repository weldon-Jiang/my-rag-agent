# 天气查询技能

## 使用场景
当用户询问天气相关问题时使用此技能。

## 使用步骤
1. 解析用户输入，获取城市名称
2. 调用天气 API 获取数据
3. 格式化结果返回给用户

## 返回格式
- 温度：摄氏度
- 湿度：百分比
- 风速：km/h
- 空气质量：AQI 指数
