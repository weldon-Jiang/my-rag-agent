# 计划：用户名称自定义功能

## 需求分析

1. **用户名称**：用户可以给自己取名字
2. **称呼用户**：智能体在对话中称呼用户的名字
3. **持久化**：用户名称保存在浏览器 localStorage

## 实现方案

### 步骤1: 前端存储用户名称
- 文件：`public/app.js`
- 添加 userName 相关的存储函数
- 存储键名：`rag_agent_user_name`
- 默认值：null 或空

### 步骤2: 前端发送用户名称
- 在发送聊天请求时，携带 userName 参数

### 步骤3: 后端添加用户改名意图
- 文件：`server/routes/chat.js`
- 添加用户改名的意图关键词
- 关键词：我叫、我是、叫我、以后叫我、叫我xxx等

### 步骤4: 提取用户新名称
- 添加 `extractUserName()` 函数
- 处理各种命名格式

### 步骤5: 修改系统提示
- 修改 `buildSystemPrompt` 函数
- 传入用户名称
- 添加用户称呼提示

### 步骤6: 用户改名响应
- 后端返回新用户名称
- 前端更新本地存储

## 技术细节

### 存储结构
```javascript
// localStorage
{
  botName: "智能助手",  // 智能体名称
  userName: "小明"       // 用户名称
}
```

### 用户改名意图检测
```javascript
rename_user: ['我叫', '我是', '叫我', '以后叫我', '以后你叫我', '你就叫我', '我的名字是', '我以后叫']
```

### 用户名称提取正则
```javascript
const userNamePatterns = [
  /我叫\s*([^\s，。,]{1,10})/,
  /我是\s*([^\s，。,]{1,10})/,
  /叫我\s*([^\s，。,]{1,10})/,
  /我的名字是\s*([^\s，。,]{1,10})/,
  /以后?我(?:就是|叫)?\s*([^\s，。,]{1,10})/,
];
```

### 系统提示增强
```
用户名称：小明

请在回答中适当称呼用户的名字"小明"，让对话更亲切。
```
