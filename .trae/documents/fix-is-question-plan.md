# 计划：修复意图检测问题

## 问题分析

从日志中发现关键问题：
1. 系统错误地将"从现在开始，你叫小通，我是你的老板。"识别为问句
2. 因为 isQuestion 返回 true，导致意图检测被完全跳过

## 原因

isQuestion 函数中的 "谁" 关键词匹配到了 "你叫小通" 中的 "叫小通"？

不对，让我仔细看...

实际上问题是 isQuestion 函数中的问句模式 `/是谁/` 会匹配到句子的任何位置。

## 解决方案

### 1. 修复 isQuestion 函数
- 更严格地匹配问句
- 只在句尾或特定位置检测问句关键词
- 排除包含"我是你的"这类陈述句

### 2. 移除调试日志（之后）
