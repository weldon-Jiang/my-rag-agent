# 知识库检索问题分析与修复计划

## 问题分析

用户输入"项目经理面试自我介绍怎么说"在知识库中找不到结果。

**原因排查**：

1. **知识库文件内容**：
   - `AGENTS.md` - 代码规范文档，无相关内容
   - `我的文档.txt` - 法国大革命历史文章，无相关内容
   - `常见项目管理面试技巧.pdf` - PDF 文件
   - `项目经理面试常遇问题.pdf` - PDF 文件
   - `简历.png` - 图片文件

2. **核心问题**：
   - 当前 `pdfs-skill.js` 只返回 PDF 文件元信息（文件名、大小）
   - **没有真正解析 PDF 文本内容**
   - 图片文件同样只返回元信息

3. **检索逻辑**：
   - 文本文件检索正常
   - PDF/图片文件因无法提取内容，所以匹配不到

## 修复方案

### 方案：添加 PDF 文本内容解析

**步骤 1**: 安装 pdf-parse 库
```bash
npm install pdf-parse
```

**步骤 2**: 更新 pdfs-skill.js
- 使用 pdf-parse 提取 PDF 实际文本内容
- 返回提取的文本内容用于检索匹配
- 处理图片文件的 OCR 提取（可选，后续优化）

**步骤 3**: 测试验证
- 上传包含"项目经理面试自我介绍"内容的 PDF
- 验证检索能否正确匹配

### 备选方案：如果 PDF 解析复杂
- 在上传 PDF 时自动提取文本存储到对应 .txt 文件
- 后续检索优先从文本内容匹配
