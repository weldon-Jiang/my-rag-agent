# 计划：调试智能体改名问题

## 问题分析

从截图看到：
1. 用户说"从现在开始，你叫小通，我是你的老板。"
2. 问"你是谁"时，回答"我叫老板"
3. 说明智能体名称没有改成"小通"，而是用了关系"老板"

## 根本原因

问题出在 buildSystemPrompt 函数中：
- 当有关系时，使用关系作为称呼
- 但没有同时使用智能体名称
- 导致智能体自我介绍时说的是关系而不是名称

## 解决方案

### 1. 添加调试日志
在 chat.js 中检测改名意图时添加日志，确认是否提取到了"小通"

### 2. 修改 buildSystemPrompt
- 智能体名称和关系是两个独立的概念
- 智能体名称用于自我介绍（"我叫小通"）
- 关系用于称呼用户（"老板，您好"）

### 3. 检查前端响应处理
确认前端是否正确保存了 newBotName
