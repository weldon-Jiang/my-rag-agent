# 修复大模型思考和检索问题

## 问题
大模型收到工具返回的结果后，没有继续思考和检索，直接返回内容给用户。

## 原因分析
1. System Prompt 指令不够明确
2. LLM 可能在模拟调用工具，但没有真正执行
3. 上下文信息没有正确传递

## 解决方案

### 1. 修改 System Prompt 指令
更明确地告诉 LLM：
- 工具已执行，结果在下方
- 需要基于结果生成完整回答
- 不要只是重复结果，要综合分析

### 2. 增强日志输出
在发送给 LLM 前，打印完整的请求内容，确认数据正确

### 3. 检查工具结果格式
确保传递给 LLM 的数据格式正确

## 实现步骤

### Step 1: 修改 buildSystemPrompt
增强指令，明确 LLM 需要基于工具结果生成回答

### Step 2: 添加调试日志
在调用 LLM 前，打印完整的 messages 内容

### Step 3: 验证数据流
确保工具结果正确传递给 LLM
