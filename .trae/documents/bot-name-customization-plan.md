# 计划：智能体名称自定义功能

## 需求分析

1. **初始名称**：智能体默认名称为"智能助手"（AIWing）
2. **改名功能**：用户可以给智能体改名，如"我叫小明"
3. **持久化**：改名信息需要保存（前端 localStorage）
4. **名称响应**：当用户询问"你叫什么"、"你是谁"等时，回答改名后的名称

## 实现方案

### 步骤1: 前端存储智能体名称
- 文件：`public/app.js`
- 使用 localStorage 存储 botName
- 默认值为"智能助手"
- 添加获取/设置名称的函数

### 步骤2: 检测改名意图
- 文件：`server/routes/chat.js`
- 在 INTENT_KEYWORDS 中添加检测改名意图的关键词
- 关键词：叫我、改名、叫、名字是、名称为、你叫等

### 步骤3: 提取新名称
- 使用正则表达式从用户输入中提取新名称
- 处理各种命名格式

### 步骤4: 更新系统提示
- 修改 buildSystemPrompt 函数
- 动态插入智能体名称
- 添加名称相关的上下文

### 步骤5: 处理名称询问
- 在 INTENT_KEYWORDS 中添加询问名称的意图
- 关键词：你叫什么、你是谁、你的名字等
- 在回答时使用存储的名称

## 技术细节

### 存储结构
```javascript
// localStorage
{
  botName: "智能助手"  // 或用户自定义的名称
}
```

### 改名意图检测
```javascript
rename_bot: ['叫我', '改名', '叫', '名字是', '以后叫你', '你就叫'],
ask_name: ['你叫什么', '你是谁', '你的名字', '名字', '叫什麼', '叫甚麼']
```

### 名称提取正则
```javascript
const namePatterns = [
  /叫我([^，。,\s]+)/,
  /以后?叫([^，。,\s]+)/,
  /名字是([^，。,\s]+)/,
  /你就叫([^，。,\s]+)/
];
```
