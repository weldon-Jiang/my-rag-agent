# 知识库管理页面优化计划

## 当前问题

1. 页面布局不合理，所有内容垂直堆叠
2. 工具说明放在页面底部，不方便查看
3. 上传区域和文件列表没有清晰分隔

## 优化方案

### 布局调整

```
┌─────────────────────────────────────────────┐
│              知识库管理                       │
├─────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────────────┐ │
│  │              │  │                      │ │
│  │  文件上传区域  │  │    已上传文件列表     │ │
│  │              │  │                      │ │
│  │  拖拽上传    │  │  - file1.pdf       │ │
│  │              │  │  - file2.jpg        │ │
│  └──────────────┘  └──────────────────────┘ │
├─────────────────────────────────────────────┤
│  🛠️ 可用工具说明 (可折叠)                    │
│  • bash  • python  • ls  • read_file...   │
└─────────────────────────────────────────────┘
```

---

## 实施步骤

### 1. 修改 HTML 结构
- 添加页面标题
- 调整上传区域和文件列表为左右布局
- 工具说明改为可折叠卡片

### 2. 修改 CSS 样式
- 添加左右分栏布局
- 优化卡片样式
- 添加响应式设计
