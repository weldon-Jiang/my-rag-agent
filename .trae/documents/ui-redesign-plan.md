# 前端 UI 改造计划 - 会话管理功能

## 目标
1. 将左侧菜单移至顶部
2. 在 AI 对话页面添加会话列表侧边栏
3. 实现会话保存功能
4. 支持会话历史展示和切换

---

## 二、改造步骤

### 2.1 步骤一：修改 HTML 结构 (index.html)

1. 将 sidebar 从 main-content 中移出
2. 在 header 中添加顶部导航栏
3. 修改 chatPage 布局，添加会话列表侧边栏
4. 添加新建会话按钮

### 2.2 步骤二：修改 CSS 样式 (style.css)

1. 创建顶部导航栏样式 `.top-nav`
2. 修改 sidebar 为 `.session-sidebar`
3. 添加会话项样式 `.session-item`
4. 添加新建会话按钮样式
5. 修改布局为水平排列

### 2.3 步骤三：修改前端逻辑 (app.js)

1. 添加会话管理状态变量
2. 添加会话列表渲染函数
3. 添加新建会话函数
4. 添加删除会话函数
5. 添加会话切换函数
6. 修改消息发送逻辑，关联会话
7. 从 localStorage 加载/保存会话

### 2.4 步骤四：后端支持（如需要）

检查现有 chat.js 是否需要添加会话相关 API

## 三、具体文件修改

### 3.1 index.html 修改

| 修改项 | 说明 |
|--------|------|
| header | 添加顶部导航栏 |
| sidebar | 改为会话列表容器 |
| chatPage | 添加会话列表侧边栏 |

### 3.2 style.css 修改

| 修改项 | 说明 |
|--------|------|
| .top-nav | 顶部导航栏样式 |
| .session-sidebar | 会话列表侧边栏 |
| .session-item | 单个会话项样式 |
| .new-session-btn | 新建会话按钮 |
| .delete-session-btn | 删除会话按钮 |

### 3.3 app.js 修改

| 函数 | 说明 |
|------|------|
| sessions | 会话列表状态 |
| currentSessionId | 当前会话 ID |
| renderSessionList() | 渲染会话列表 |
| createNewSession() | 创建新会话 |
| deleteSession(id) | 删除会话 |
| switchSession(id) | 切换会话 |
| saveSession() | 保存会话到 localStorage |
| loadSessions() | 从 localStorage 加载会话 |

## 四、数据结构

```javascript
{
  id: "session_1234567890",
  title: "会话标题",
  createdAt: 1234567890,
  updatedAt: 1234567890,
  messages: [
    {
      role: "user",
      content: "用户问题",
      timestamp: 1234567890
    },
    {
      role: "assistant",
      content: "AI回答",
      timestamp: 1234567890,
      source: "ai",
      knowledgeResults: []
    }
  ]
}
```

### 4.2 localStorage 存储

- 键名: `rag_agent_sessions`
- 值: JSON 字符串化的会话数组
