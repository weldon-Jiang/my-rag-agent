# 模型列表状态操作功能计划

## 需求分析
在模型列表卡片中直接显示状态，并提供快捷切换按钮，无需打开编辑弹窗即可切换发布状态。

## 修改内容

### 1. 修改 renderModelsList 函数
在模型卡片中添加状态切换按钮：
- 显示当前状态（已发布/未发布）
- 添加"切换状态"按钮
- 调用 API 快速更新状态

### 2. 添加 toggleModelPublished 函数
- 接收模型 ID
- 发送 PUT 请求更新发布状态
- 更新成功后刷新列表

## 实现步骤

### 步骤 1: 修改 renderModelsList
- 在模型卡片操作区添加状态切换按钮
- 按钮文字根据当前状态显示

### 步骤 2: 添加 toggleModelPublished 函数
- 实现状态切换逻辑
- 调用后端 PUT /api/models/:id 接口

### 步骤 3: 测试验证
- 验证状态显示正确
- 验证点击切换按钮状态正确更新
