# 地理位置 Skill

## 使用场景
当用户询问地理位置相关问题时使用：
- "北京是哪个省的"
- "深圳在哪里"
- "经纬度是多少"

## 工作流程
1. 解析用户问题中的地名
2. 调用地理编码 API 获取坐标
3. 调用反地理编码获取行政区划信息

## 参数说明
- `location`: 地名（如"北京"、"深圳市"）

## 注意事项
- 如果地名模糊，需要追问确认
- 返回结果包含经纬度和行政区划信息
