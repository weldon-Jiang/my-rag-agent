# Command-Skill

命令处理技能，识别和处理用户的系统命令请求。

## 支持的命令类型

### clear - 清空历史
- 关键词：清空、清算、清一下、清理、重置历史、删除历史
- 响应：已清空对话历史

### reset - 重置会话
- 关键词：重新开始、重启、重置、新对话、新会话
- 响应：已重置会话

### help - 帮助信息
- 关键词：帮助、help、怎么用、如何使用、说明、功能
- 响应：显示帮助信息

### exit - 退出对话
- 关键词：退出、exit、关闭、再见、拜拜
- 响应：再见
