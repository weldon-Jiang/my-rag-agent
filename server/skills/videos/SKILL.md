# 视频内容理解 Skill

## 使用场景

当用户上传视频并需要分析内容时使用：

- "这个视频讲了什么"
- "帮我分析视频"
- "视频里有什么"

## 工作流程

1. 提取视频关键帧
2. 分析视频场景和内容
3. 返回关键信息摘要

## 参数说明

- `filename`: 视频文件名
- `filepath`: 视频文件路径

## 支持格式

- .mp4, .avi, .mov, .mkv, .webm, .flv, .wmv

## 注意事项

- 视频需要先上传到服务器
- 分析可能需要较长时间
