# 天气查询 Skill

## 使用场景
当用户询问天气相关问题时使用：
- "今天天气怎么样"
- "明天会下雨吗"
- "北京现在多少度"

## 工作流程
1. 解析用户问题中的城市和时间
2. 调用天气 API 获取数据
3. 格式化回复内容

## 参数说明
- `query`: 用户查询（如"北京天气"）
- `context.originalQuery`: 原始问题（用于提取日期）

## 注意事项
- 如果用户没指定城市，需要追问
- 温度需要说明是摄氏还是华氏
- 日期解析优先级：originalQuery > query
