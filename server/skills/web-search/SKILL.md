# 网页搜索 Skill

## 使用场景
当用户需要在互联网上搜索信息时使用：
- "帮我搜索一下 xxx"
- "网上怎么说的"
- "查一下 xxx"

## 工作流程
1. 提取用户查询的关键词
2. 调用搜索引擎 API
3. 返回搜索结果

## 参数说明
- `query`: 搜索关键词
- `max_results`: 最大结果数（默认 5）

## 注意事项
- 关键词需要精简
- 返回结果包含标题、链接、摘要
